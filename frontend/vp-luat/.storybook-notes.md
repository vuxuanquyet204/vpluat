# Storybook Integration Notes

Dự án đã thêm script `storybook` và `build-storybook` trong `package.json` để chốt scope Phase 2 ở mức skeleton.

Hiện tại chưa cài được package Storybook vì ecosystem Storybook public stable chưa hỗ trợ peer dependency cho `next@16.2.6` trong workspace này.

Khuyến nghị khi triển khai thật:
- chờ adapter Storybook tương thích `Next 16`
- hoặc tách Storybook sang Vite-only preview cho component thuần
- hoặc hạ / pin version Next nếu bắt buộc phải dùng Storybook ngay
